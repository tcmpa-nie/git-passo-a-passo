# Uso do GIT : Passo a Passo (git-passo-a-passo)

Este repositório contém dicas de uso de comandos básicos para você usar o GIT no seu dia a dia, criando um fluxo de trabalho simplificado, porém que se adapta à maioria das tarefas de um _sprint_ de uma nova versão de software.

Para o uso adequado destes comandos, considera-se que seu repositório de trabalho encontra-se no GITHUB e que você tenha acesso a um terminal de comandos no seu computador ( bash, cmd, powershell, etc.) e também tenha instalado o GIT. Os comandos se aplicam a todos os sistemas operacionais mais usados: Windows, MacOS e Linux.

O objetivo é que em um curto espaço de tempo você possa usar estes comandos para incluir/alterar código em seu repositório de trabalho, possibilitando compartilhar com os outros membros de sua equipe.
